# vue3-components

## ‍💻 技术栈

- 开发框架：Vue.js（3）
- 开发语言：JavaScript
- 前端工程化：Webpack
- 语法拓展：Tsx、Scss、TypeScript
- 表单验证：Async-Validator
- 语法检测：ESLint、Stylelint、Prettier

## ✨ 项目介绍

该项目封装了一些组件，包含了一些常用的组件，例如：

- Form 表单
- FormItem 表单项
- Input 输入框

### 主要特性

- provide/inject（依赖注入）
- tsx基本使用（之前并没有用过）
- expose（向外暴露属性）
- 父子组件互相通信
- 表单验证（async-validator）
